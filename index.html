What is Zintillex

A file format built
for 3D artists.

Zintillex is a new open 3D bundle format. Unlike FBX or OBJ, a .zint file embeds everything — geometry, metadata, assets — directly inside a single portable file. No broken links. No missing textures. Just one file.

01

Self-contained

All 3D data is encoded and embedded inside the .zint file. Send it anywhere and it just works.

02

Blender native

Import and export .zint files directly from Blender 4.x with the official add-on.

03

Open format

Plain text based. Human readable. Anyone can build tools that read and write .zint files.

04

Viewer app

The Zintillex app lets you browse, preview, and manage your .zint bundles on Windows and Linux.

How it works

Simple by design.

1

Model in Blender

Create your 3D model in Blender as you normally would.

2

Export as .zint

Use File → Export → Zintillex (.zint) to bundle everything into one file.

File → Export → Zintillex (.zint)
3

Share it

Send the single .zint file to anyone. They can open it in the Zintillex viewer or import it back into Blender.

4

Or use the CLI

Power users can create and manage .zint bundles directly from the terminal.

zint new MyProject
